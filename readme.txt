== Short Description ==

Hide the WordPress admin bar for non-admin users, providing a cleaner experience for contributors, authors, and editors.

== Description ==

Hide the WordPress admin bar for non-admin users. This plugin allows you to hide the admin bar from all users except administrators, providing a cleaner experience for subscribers, contributors, authors, and editors.

== Installation ==

1. Upload the `achi-hide-admin-bar` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The admin bar will be hidden automatically for all non-admin users.

== Frequently Asked Questions ==

= Does this plugin hide the admin bar for all users? =

No, it only hides the admin bar for users who do not have the 'administrator' role.

= Can I customize which roles see the admin bar? =

This version of the plugin is designed to hide the admin bar for all non-admin users by default. Customization can be done by editing the plugin code.
